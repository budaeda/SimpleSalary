# Configuration
``` xml
<?xml version="1.0" encoding="utf-8"?>
<SimpleSalaryConfiguration xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <SalaryInfoList>
    <SalaryInfo>
      <PermissionsGroupId>default</PermissionsGroupId>
      <SalaryPayout>1000</SalaryPayout>
    </SalaryInfo>
    <SalaryInfo>
      <PermissionsGroupId>police</PermissionsGroupId>
      <SalaryPayout>5000</SalaryPayout>
    </SalaryInfo>
  </SalaryInfoList>
  <SalaryPayoutInterval>5</SalaryPayoutInterval>
</SimpleSalaryConfiguration>
```
